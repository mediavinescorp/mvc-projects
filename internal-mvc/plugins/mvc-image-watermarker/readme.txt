Batch watermark images with logo + dual text overlays. Outputs JPEG with structured SEO filenames.

== Description ==

A browser-based batch image tool for home service contractors and local businesses.
Drop in any images, add your logo and up to 2 text overlays, set output dimensions, and download
all files as JPEG — without uploading anything to the server.

**File naming modes:**
- Structured: Industry - Keyword - City - State - Client (e.g. HVAC-heatpump-northridge-ca-socal-climate-control.jpg)
- Add prefix
- Rename each manually
- Keep original filename

**Usage:** Add `[image_watermarker]` to any page or Elementor HTML widget.

**Settings:** WordPress Admin → Settings → MVC Watermarker

== Installation ==

1. Upload the `mvc-image-watermarker` folder to `/wp-content/plugins/`
2. Activate the plugin through the Plugins menu
3. Go to Settings → MVC Watermarker to set your defaults
4. Add `[image_watermarker]` to any page
